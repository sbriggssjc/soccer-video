# Tulsa Soccer Club Brand Kit

This folder contains the Tulsa Soccer Club (TSC) overlays and reference assets used by the automated branding scripts.

## Colors
- **Navy** `#1F2B3D`
- **Red** `#9B1B33`
- **Gold** `#B7A37C`
- **White**
- **Black**

## Typography
- Headlines and slates use **Montserrat** (ExtraBold, Bold, SemiBold).
- Body copy and supporting text use **Inter** or **Roboto**.

## Watermark Rules
Place the watermark in the bottom-left corner with 32px padding from the edges at 65% opacity.

## Safe Area
Keep essential action and text inside a 5% inset on all sides for both landscape (16:9) and portrait (9:16) layouts.
